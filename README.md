OIBSIP (Oasis Infobyte Internship Projects)

Welcome to my GitHub repository for the Oasis Infobyte Virtual Internship program. This repository contains the Python projects I developed during my internship, showcasing practical applications of programming concepts.

Projects Overview

1. BMI Calculator

-The Body Mass Index (BMI) Calculator is a tool designed to help users monitor their health status.

-It takes user input for weight in kilograms and height in meters or centimeters

-It calculates the BMI value using standard mathematical formulas

-It categorizes the result into health zones like underweight, normal weight, overweight, or obese

-It provides a clean command-line interface for ease of use

2. Simple Random Password Generator

-The Random Password Generator is a security-focused utility that builds strong and secure passwords.

-It allows users to specify their preferred length for the password

-It mixes uppercase letters, lowercase letters, digits, and special characters

-It ensures the generated passwords are completely randomized and difficult to guess

-It helps users enhance their digital security across various accounts

3. Voice Assistant

-The Voice Assistant is an interactive program capable of understanding and executing verbal commands.

-It utilizes advanced AI tools to accelerate the learning and implementation of key Python libraries

-It incorporates the speechrecognition library to capture and process human voice input

-It integrates the pyttsx3 library to provide text-to-speech feedback for a natural conversation flow

-It performs basic automated tasks like opening websites or stating the current time based on voice prompts
